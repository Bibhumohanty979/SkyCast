# SkyCast : Weather Forecast App

Intern ID: CITS6090

No. of Weeks: 4 Weeks

Project Name: SkyCast – Weather Forecast App

SkyCast – A weather forecast application that provides real-time weather information and forecasts for cities around the world. Built using HTML, CSS, and JavaScript. Users can search for cities, check current weather conditions, view hourly and 7-day forecasts, use their current location, switch between Celsius and Fahrenheit, and use light or dark mode.

# Features

🔍 City Search

📍 Current Location Weather

🌤️ Current Weather Conditions

🌡️ Temperature & Feels-Like Temperature

💧 Humidity Information

💨 Wind Speed

👁️ Visibility & Atmospheric Pressure

🌅 Sunrise & Sunset Information

⏰ Hourly Weather Forecast

📅 7-Day Weather Forecast

🌡️ Celsius & Fahrenheit Switching

🕘 Recent Search History

🌙 Light & Dark Mode

📱 Responsive Design

💾 Local Storage Support

# 🛠️ Technologies Used

HTML5

CSS3

JavaScript

Open-Meteo Weather API

# 📌 Project Purpose

This project demonstrates the development of an interactive and responsive weather forecast application using frontend web technologies and weather API integration. It provides users with an easy-to-use interface for accessing current weather conditions and future forecasts for different locations.
